BDX-R

A low-cost bipedal robot inspired by Disney's BD-X droid.

About the Project

After first seeing Disney showcase their BD-X droids, I knew I had to make one myself. I had never built a robot with reinforcmenet learning before, and so I decided to make this project my Bachelor End Project (BEP), a 16 week project at my university (TU/e). I quickly realized that there was no cler path for someone to follow on how to make a robot learn to walk with reinforcement learning and deploy it onto physical hardware. I solved this problem by creating a blueprint that anyone can follow if they are looking to make a bipedal robot learn to walk with RL. I was awarded a 9/10 for my project and was able to train the robot in simulation with NVIDIA's IsaacLab and deploy it onto the physical hardware. At the end of my project, I was able to see the robot take its first steps in the real world. This project is what ultimately sparked my passion for using RL to make robots walk. For the timespan of the project, I decided to exclude the head of the robot as it was unesecarry for locomotion, however I am working on currently adding it. The BDX-R, or the BDX-Robstride is the robot that I created with 10 Robstride actuators and a NVIDIA Jetson Orin Nano.
